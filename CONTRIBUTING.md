# Contributing Guidelines

## Branch Naming
- `feature/collector-dashboard`
- `feature/sarpanch-dashboard`
- `feature/officer-dashboard`
- `feature/driver-app`
- `feature/ml-prediction`
- `fix/route-optimization`

## Commit Message Format
```
feat: add collector dashboard rainfall chart
fix: tanker allocation priority bug
docs: update API spec
chore: upgrade dependencies
```

## PR Process
1. Branch from `main`
2. Make changes
3. Test locally
4. Open PR with description
5. Get 1 review approval
6. Merge
